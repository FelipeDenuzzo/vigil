# Labirintos Prolongados

Este módulo implementa o jogo "Labirintos Prolongados", com foco na avaliação da **atenção sustentada**. O usuário deve navegar por um ou mais labirintos por períodos estendidos, testando a capacidade de manter o foco e identificar rotas sem perder o objetivo.

## Arquivos

- `LabirintosProlongadosGame.tsx`: Componente principal do jogo.
- `types.ts`: Tipos básicos para a estrutura e configuração do jogo.
- `levels.ts`: Definições dos níveis do labirinto.
- `logic.ts`: Lógica pura de geração de labirintos e cálculos do jogo.

O jogo está atualmente isolado via feature flag.
